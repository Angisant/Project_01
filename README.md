# Ultrasonic Distance Sensor with LED Warning

This project uses an HC-SR04 ultrasonic sensor to measure distance.  
If an object is closer than 10 cm, an LED turns on as a warning.

## Components
- Arduino Uno
- Jumper wires, breadboard
- PushButton
- Blue LED
- 220 Î© Resistor
